# Cinematic Value and Planar Composition
Last_Updated: 2026-05-23

## Overview
- Purpose: 将电影明度、平面构成和空间分层的原理整理为跨摄影、概念设计和 AIGC 的视觉控制规范。
- Scope: Covers 明度心理学、Figure-Ground、平面分析、Notan、低调摄影、空气透视和大师案例抽象。Does not cover 单片逐镜头详解或软件操作教程。

## Core Concepts
- **明度语法**: 画面最先被感知的是大明暗关系，明度决定可读性、空间和情绪。
- **Figure-Ground**: 主体必须从背景中通过明度、边缘、色彩或形状分离出来。
- **平面分离**: 前景、中景、背景各自拥有不同明度组，形成空间秩序。
- **Notan**: 将画面压缩为黑白大块后仍成立的构图骨架。
- **Carlson角度理论**: 不同朝向的平面因受光角度不同产生明度差，可用于组织自然场景。
- **低调摄影**: 以大面积暗部和受控高光建立戏剧性，但高光位置必须服务主体。

## Consolidated Principles
- P1 [Critical]: 电影感首先是明度组织，不是 LUT、颗粒或镜头光斑。
- P2 [Critical]: 主体与背景明度过近会破坏可读性，即使细节丰富也会失败。
- P3 [High]: 大明度块必须先成立，细节只能服从大块关系。
- P4 [High]: 空间深度来自平面之间的明度、对比、边缘和饱和差异。
- P5 [High]: 低调画面不能等于全黑；暗部仍需层次和方向性。
- P6 [Normal]: 明度节奏应在主体焦点处最强，在次要区域弱化。

## Mechanisms & Logic
- Flow: 1. 将画面压缩为 Notan -> 2. 确认主体剪影 -> 3. 划分前中后景平面 -> 4. 分配明度组 -> 5. 加入边缘和色彩 -> 6. 控制细节密度。
- Decision Rules: 如果主体不突出，调整主体背后的明度，而不是只给主体加细节。
- Decision Rules: 如果空间不深，降低远景对比并强化前景暗形或中景光形。
- Decision Rules: 如果低调画面脏，抬出关键暗部层次并限制高光面积。
- Decision Rules: 如果构图散，回到黑白缩略图检查大块关系。

## Constraints
- 不要用饱和色替代明度分离。
- 不要在所有区域平均加细节和锐化。
- 不要让局部高光抢走主体焦点。
- 不要忽略显示设备和输出尺寸对暗部可读性的影响。
