# 英雄联盟原画视觉设计演变分析
Last_Updated: 2026-05-23

## Overview
- Purpose: 提炼 LoL 原画从功能性肖像到电影化叙事的视觉演变，并抽象为构图、透视、光影和区域色彩规则。
- Scope: Covers 视觉断代、黄金螺旋、三角构图、广角透视、区域色彩、摄影转译和现代原画语言。Does not cover 官方 IP 复刻或逐张作品考据。

## Core Concepts
- **功能性肖像期**: 早期更重视角色识别，构图直接、姿态相对静态。
- **动态叙事期**: 中期开始用动作、镜头和背景讲述角色能力与世界观。
- **电影化沉浸期**: 现代原画强调透视冲击、体积光、空间层次和叙事瞬间。
- **视觉引导线**: 武器、肢体、特效、建筑和光束共同指向面部或技能核心。
- **区域色彩语义**: 德玛西亚偏白金蓝；诺克萨斯偏黑红铁灰；祖安偏毒绿紫；暗影岛偏青绿黑；皮尔特沃夫偏金铜蓝；虚空偏紫洋红电蓝。
- **摄影转译**: 原画语言可转为低机位、广角、边缘光、烟雾、色片和分层布光。

## Consolidated Principles
- P1 [Critical]: LoL 风格不是单个颜色或特效，而是角色可读性、叙事姿态、透视冲击和区域语义的组合。
- P2 [Critical]: 视觉演变不能被简化为画质提升；核心变化是从识别图到叙事镜头。
- P3 [High]: 现代原画常用广角近景、短缩和低机位放大英雄感。
- P4 [High]: 色彩必须服务阵营、情绪和空间层级，而不是任意高饱和。
- P5 [High]: 摄影应用应借鉴构图逻辑，不应复制官方角色或皮肤设计。
- P6 [Normal]: 背景复杂度随叙事需求增加，但主体剪影始终优先。

## Mechanisms & Logic
- Flow: 1. 确定角色阵营或情绪 -> 2. 选择区域色彩语义 -> 3. 设计姿态和视角 -> 4. 建立视觉引导 -> 5. 分配光影层级 -> 6. 检查可读性和原创性。
- Decision Rules: 要英雄感 -> 使用低机位、广角前景和强轮廓光。
- Decision Rules: 要区域识别 -> 使用色彩和材质符号，而不是直接引用官方元素。
- Decision Rules: 要摄影转译 -> 用布光、镜头和道具表达原画逻辑。

## Constraints
- 不要直接复制 LoL 英雄、皮肤、标志或受保护视觉元素。
- 不要把区域色彩当作固定色卡，色彩还需服从光源和场景。
- 不要让特效噪音压过主体动作和面部。
- 不要忽略裁切和缩略图识别。
