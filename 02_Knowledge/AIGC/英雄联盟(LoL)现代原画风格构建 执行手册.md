# 英雄联盟(LoL)现代原画风格构建 执行手册
Last_Updated: 2026-05-23

## Overview
- Purpose: 将 LoL 现代原画的构建要素转为可执行的视觉规范，用于原创角色图像生成、概念设计和摄影转译。
- Scope: Covers 动态横向姿态、激进透视、构图引导、电影级布光、Notan、材质、魔法特效、大气透视和后期焦点控制。Does not cover 官方 IP 复制、皮肤复刻或平台绕过。

## Core Concepts
- **动态横向姿态**: 角色身体、道具和特效横向展开，适配宽画幅并制造速度。
- **激进透视**: 低机位、广角和短缩强化力量，但需要保持解剖可信。
- **构图引导**: 肢体、武器、披风、光束和背景结构共同导向脸部与技能核心。
- **Notan设计**: 大明暗结构先成立，再进入细节渲染。
- **材质区分**: 金属、布料、皮肤、皮革、魔法能量必须有不同高光和边缘行为。
- **实体化魔法**: 魔法特效应有体积、光源、遮挡和空间位置。

## Consolidated Principles
- P1 [Critical]: 必须构建原创角色和原创视觉符号，不能调用具体 LoL 英雄或皮肤复刻。
- P2 [Critical]: 动态姿态、透视、光影、材质和特效必须服务同一叙事瞬间。
- P3 [High]: 先做 Notan 和剪影，再做区域细节和粒子。
- P4 [High]: 激进透视要控制手脚、武器和脸部的优先级。
- P5 [High]: 魔法特效必须照亮周围材质并参与空间遮挡。
- P6 [Normal]: 后期光学缺陷和调色只用于统一画面，不应掩盖结构问题。

## Mechanisms & Logic
- Flow: 1. 设定原创角色身份 -> 2. 设计横向动作线 -> 3. 选择低机位和短缩 -> 4. 建立 Notan -> 5. 分层布光 -> 6. 区分材质 -> 7. 添加体积特效 -> 8. 用大气透视和焦点控制收束。
- Decision Rules: 画面缺少现代原画感 -> 加强动作线、剪影、轮廓光和材质分离。
- Decision Rules: 特效平面化 -> 增加厚度、光源、投影和遮挡。
- Decision Rules: 画面杂乱 -> 回到三层空间和一个主焦点。
- Decision Rules: 角色像复制品 -> 删除专属符号，改为抽象材质、姿态和色彩原则。

## Constraints
- 不要复刻官方英雄、阵营标志、专属武器或皮肤视觉元素。
- 不要让多个技能核心同时争夺焦点。
- 不要在未建立明度结构前堆粒子和光效。
- 不要让后期锐化、色差和 Bloom 破坏主体可读性。
