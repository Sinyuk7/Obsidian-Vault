# 英雄联盟原画风格主控系统
Last_Updated: 2026-05-23

## Overview
- Purpose: 将 LoL 现代原画风格抽象为可控的提示词与视觉设计系统。
- Scope: Covers 风格化写实、叙事沉浸、双重裁切、受控失衡、广角英雄视角、剪影、材质、光影和特效层级。Does not cover 官方角色复刻或商标使用。

## Core Concepts
- **风格化写实**: 保留可信解剖、材质和光影，同时夸张姿态、比例、透视和特效。
- **叙事性沉浸**: 画面应像战斗或剧情中的关键瞬间，而非静态设定图。
- **双重裁切约束**: 角色必须适配横幅展示、缩略图和局部焦点裁切。
- **受控失衡**: 动态姿态和倾斜构图带来能量，但明度和焦点仍需稳定。
- **广角侵略性**: 近大远小和短缩增强英雄感，但必须控制肢体变形。
- **空间层级分离**: 前景特效、中景角色、背景世界观需要清晰分工。

## Consolidated Principles
- P1 [Critical]: LoL 风格主控必须先锁定原创角色身份和视觉逻辑，不能复刻官方 IP。
- P2 [Critical]: 剪影优先于细节，缩略图不可读时所有渲染都失效。
- P3 [High]: 动态线、广角、低机位和对角线构图共同建立英雄感。
- P4 [High]: 材质区分必须清楚，皮肤、金属、布料、魔法和环境不能糊成一层。
- P5 [High]: 特效应有实体化体积和光源逻辑，而不是平面粒子堆叠。
- P6 [Normal]: 背景叙事要支持角色，不要让世界观信息抢主体。

## Mechanisms & Logic
- Flow: 1. 定义原创角色身份 -> 2. 设定叙事瞬间 -> 3. 选择英雄视角 -> 4. 建立剪影和动作线 -> 5. 分层光影材质 -> 6. 添加特效和环境 -> 7. 检查裁切。
- Decision Rules: 如果画面不够 LoL 感，优先强化姿态、短缩、轮廓光和材质对比。
- Decision Rules: 如果像普通奇幻插画，增加战斗瞬间、技能核心和广角空间冲击。
- Decision Rules: 如果像官方复刻，删除英雄名、标志性装备和专属符号，保留抽象风格原则。

## Constraints
- 不要使用受保护英雄名称、皮肤名称、阵营标志或专属装备复刻。
- 不要让效果覆盖脸部和主动作。
- 不要同时堆多个主光源、多个技能核心和多个动作方向。
- 不要牺牲解剖可信度追求广角冲击。
