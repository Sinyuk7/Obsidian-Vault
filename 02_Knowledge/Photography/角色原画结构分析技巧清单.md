# 角色原画结构分析技巧清单
Last_Updated: 2026-05-23

## Overview
- Purpose: 将角色原画结构分析整理为可复用的观察清单，帮助从平面图像还原透视、体块、姿态和材质逻辑。
- Scope: Covers 视平线、焦距、消失点、体块化、躯干模型、肢体关节、衣褶装备和综合检查。Does not cover 官方角色复刻或医学解剖诊断。

## Core Concepts
- **结构分析**: 从原画中识别支撑姿态、空间透视、体块连接和材质层级，而不是只临摹外形。
- **视平线定位**: 通过眼高、背景线、椭圆开口和身体透视判断观者高度。
- **焦距感**: 广角增强近大远小和冲击力，长焦压缩空间并强化平面设计。
- **Mannequinization**: 将角色还原为可旋转的胸腔、骨盆、头颈、四肢和关节体块。
- **Bean / Robo Bean**: Bean 强调躯干流动，Robo Bean 强调胸腔与骨盆的盒体方向。
- **结构清单**: 透视、体块、重心、肢体、衣褶、装备和光影应逐项检查。

## Consolidated Principles
- P1 [Critical]: 分析角色原画必须先找三维结构，再评价线条和细节。
- P2 [Critical]: 胸腔、骨盆和头部方向决定姿态可信度。
- P3 [High]: 消失点和视平线不必完美严格，但必须形成可解释的空间一致性。
- P4 [High]: 肢体短缩要通过截面、重叠和体积体现，不是简单缩短线段。
- P5 [High]: 衣褶和装备应依附体块和受力点，不能独立漂浮。
- P6 [Normal]: 分析清单应服务重建，不应变成术语罗列。

## Mechanisms & Logic
- Flow: 1. 判断视平线和机位 -> 2. 判断焦距感 -> 3. 找主动作线 -> 4. 标胸腔骨盆头部 -> 5. 还原四肢体块 -> 6. 检查装备衣褶 -> 7. 验证重心和光影。
- Decision Rules: 如果姿态不可信，先修胸腔骨盆关系。
- Decision Rules: 如果肢体不读，增加关节体块和截面方向。
- Decision Rules: 如果装备混乱，找挂点、厚度和遮挡关系。
- Decision Rules: 如果画面平，检查前中后景和明度分组。

## Constraints
- 不要直接复制受保护角色设计。
- 不要把装饰细节当成结构依据。
- 不要忽略被遮挡肢体的空间延续。
- 不要让分析停留在外轮廓相似。
