# 英雄联盟原画风格的摄影应用
Last_Updated: 2026-05-23

## Overview
- Purpose: 将 LoL 原画的构图、透视、光影、材质和色彩逻辑转译为摄影拍摄与后期指导。
- Scope: Covers 动态构图、广角英雄视角、低机位、体积光、边缘光、冷暖对冲、材质强化和后期电影化。Does not cover 复制官方角色、服装或受保护视觉资产。

## Core Concepts
- **原画摄影化**: 用真实镜头、布光、姿态和后期模拟原画的英雄感，而不是复制插画表面。
- **动态三角**: 头、手、武器或道具构成稳定但有冲击的三角关系。
- **英雄视角**: 低机位、广角近距离和透视短缩让人物更具力量感。
- **体积光**: 烟雾、逆光和侧逆光形成空间层次与史诗感。
- **材质可读性**: 服装、皮肤、金属、皮革和特效道具需要不同高光和纹理处理。
- **冷暖对冲**: 暖肤色或主光与冷背景或边缘光分离主体。

## Consolidated Principles
- P1 [Critical]: 摄影应用应转译 LoL 原画的视觉原则，而不是复刻具体 IP。
- P2 [Critical]: 姿态和镜头先决定英雄感，后期特效只能增强不能替代。
- P3 [High]: 低机位广角要控制面部和肢体变形，不能让近端手脚抢主体。
- P4 [High]: 光影层级必须让面部、胸腔和主道具优先可读。
- P5 [High]: 服装材质和道具边缘需要高光分离，避免黑成一团。
- P6 [Normal]: 背景烟雾、粒子和光束应服务空间，不应变成噪点。

## Mechanisms & Logic
- Flow: 1. 设计姿态和动作线 -> 2. 选择低机位或广角程度 -> 3. 安排主光、轮廓光和背景光 -> 4. 加烟雾或体积介质 -> 5. 拍摄材质高光 -> 6. 后期加强明度分层和色彩对冲。
- Decision Rules: 画面不够英雄 -> 降低机位、强化对角线和轮廓光。
- Decision Rules: 人像变形过强 -> 后退机位并使用较长焦段或后期局部校正。
- Decision Rules: 材质不读 -> 增加边缘光、偏振控制或局部 Dodge & Burn。
- Decision Rules: 背景抢戏 -> 降低背景饱和和锐度。

## Constraints
- 不要使用官方英雄服装、标志或专属武器作为可识别复刻。
- 不要让后期光效遮挡真实姿态和面部表情。
- 不要用一层 LUT 代替布光和明度设计。
- 不要忽略拍摄阶段的安全和道具可控性。
