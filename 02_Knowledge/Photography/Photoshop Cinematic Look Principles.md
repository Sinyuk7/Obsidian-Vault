# Photoshop 中实现电影感的思路

## Overview

* **Purpose**: 提炼 Photoshop 中实现电影感与 CG/原画感的核心思路、关键概念与方法框架。
* **Scope**: Covers 视觉逻辑、光学与胶片线索、色彩与影调、空间深度、材质与合成思维。Does not cover 详细按钮路径、逐步教程式 SOP、弱相关历史背景。

## Core Concepts

* **电影感**: 不是单一滤镜，而是对真实摄影系统痕迹的重建。核心是 **自然记录质感 + 受控风格化**，让画面看起来像经由真实镜头、胶片或电影级后期产生。
* **CG感 / 原画感**: 不是单一锐化或高对比，而是 **结构化光影、材质可读性、形体层级、空间秩序与可设计性** 更强的画面组织方式。
* **受控的瑕疵**: 电影感的核心线索之一。通过模拟色差、泛光、光晕、暗角、颗粒、镜头污渍、漏光、畸变等缺陷，打破数字图像过于干净、过于锐利、过于无机的状态。 
* **影调塑形**: 通过高光过渡、高光压缩、黑位抬升、中间调对比、局部明暗雕刻，决定画面的“电影底子”。 
* **色彩组织**: 通过冷暖分离、分区色调、LUT、通道曲线、Selective Color、Gradient Map 等手段建立情绪与空间关系。 
* **空间深度线索**: 通过景深、空气透视、AO、雾化、前中后景分层、Z-Depth 思维，让二维图像产生三维体积感。 
* **材质与表面物理属性**: CG/原画感高度依赖高光形状、粗糙度、边缘光、次表面透射感、微观纹理与频率分离后的材质控制。 
* **非破坏式 Pass 思维**: 稳定风格来自可回滚的分层流程，而不是一次性把效果做死。典型顺序是技术底片层 → 影调层 → 色彩层 → 镜头/胶片层 → 合成统一层。 

## Consolidated Principles

> [!danger] Critical 原则
> - **P1**：电影感与 CG 感都不是滤镜名，而是多组视觉线索的协同结果。单独强化某一项通常不足以成立整体风格。
> - **P2**：先建立结构，再叠加风格。优先顺序是影调与体积、再到色彩、最后才是颗粒、光晕、色差等表层痕迹。
> - **P3**：数字图像的问题通常不是“不够强烈”，而是“过于完美、过于平、过于干净”。电影感依赖受控地破坏这种完美。#critical-principles

> [!warning] High 原则
> - **P4**：电影感偏向“镜头与介质的真实性”，CG 感偏向“结构、材质、分层与设计可读性”；二者可叠加，但主导逻辑不同。
> - **P5**：颜色不是装饰，而是叙事工具。冷暖对比、主体与背景分离、肤色保护、环境色统合，比单纯提高饱和度更重要。
> - **P6**：Photoshop 中最重要的不是“按钮”，而是把光学、胶片、视觉心理学和 3D 渲染逻辑翻译成图层、通道、混合模式与蒙版。#color-grading

> [!tip] 平衡点
> **P7 [Normal]**：最终质感往往来自“构建数学级清晰结构”与“叠加物理级不完美”这两步的平衡。

## Mechanisms & Logic

* **总流程**

  1. 先校正与保留动态范围
  2. 再建立影调结构
  3. 再组织色彩关系
  4. 再补镜头与胶片痕迹
  5. 最后统一合成、锐度、噪点与输出观感。
* **电影感为什么成立**

  * 因为它复现了人对电影媒介的长期视觉记忆：高光不过硬、暗部不死黑、边缘不是绝对干净、亮部会扩散、颜色会分区、镜头会留下瑕疵。 
  * 关键影调特征包括：

    * **Highlight Roll-off**: 高光接近白时平滑过渡，而不是突然剪切。
    * **Lift Blacks**: 轻抬黑位，制造胶片式底灰与呼吸感。
    * **Midtone Contrast**: 主要靠中间调与局部对比塑形，而不是暴力全局对比。 
  * 关键光学/胶片特征包括：

    * **Chromatic Aberration**: 边缘通道轻微错位，暗示真实镜头存在。
    * **Bloom**: 极亮区域向外扩散，降低数字边缘的生硬感。
    * **Halation**: 高亮边缘出现暖红侵蚀感，强化胶片气质。
    * **Vignette / Dirt / Leak / Flare**: 强化“通过镜头观看”的媒介感。 
  * 关键色彩逻辑包括：

    * **Teal and Orange / Split Toning**: 用冷暖对立拉开主体与环境。
    * **LUT / 通道曲线 / Selective Color**: 用于全局定调与局部保护。
    * **Log / ACES 思维**: 先保留宽容影调，再做风格输出。 
* **CG感 / 原画感为什么成立**

  * 因为它强化了现实中不总是如此清晰的结构关系，让物体“可读”“可设计”“可分层”。
  * 关键结构线索包括：

    * **AO 思维**: 强化接触阴影与缝隙遮蔽，给物体“落地感”和重量。
    * **Z-Depth / 景深 / 空气透视**: 用距离控制模糊、对比、饱和与冷暖，让空间更像 3D 场景。
    * **FG/MG/BG 分层**: 前景、中景、背景各自拥有独立的对比、锐度和色彩密度。  
  * 关键材质线索包括：

    * **Specular 设计**: 高光形状决定材质类别。
    * **Roughness 感知**: 高光边缘软硬决定表面粗糙度。
    * **Rim Light**: 提升轮廓分离与形体读取。
    * **局部透光 / SSS 感**: 让薄处更暖更透，增强生物或半透明材质的可信度。
  * 关键原画/绘画化线索包括：

    * **Gradient Map** 统一大气色盘。
    * **值域分组 / Notan 思维** 强调大明暗块而非碎细节。
    * **Color Jitter** 与有机色彩波动可打破纯数字平滑感，形成绘画肌理。 
* **电影感与 CG 感如何结合**

  * 先用 CG/原画逻辑把结构、空间、材质做“清楚”。
  * 再用电影逻辑把镜头、胶片、色彩与高光行为做“有机”。
  * 结果不是二选一，而是“结构清晰的电影感”或“带镜头媒介痕迹的 CG 感”。 

## Constraints

> [!warning] 高风险误区
> - **不要**把电影感理解成单纯加 LUT、加暗角、加黑边。这样只会得到表面模仿，得不到影调、光学和介质层面的可信度。
> - **不要**把 CG 感理解成单纯锐化、提清晰度、拉高对比。这样容易出现塑料感、脏边和过度数字化。
> - **不要**先堆表层特效，再补结构。顺序反了会导致画面从底层就不成立。
> - **不要**全图平均施加色差、Halation、Bloom、漏光、镜头污渍。真实情况通常只在特定亮度区间、边缘区域、光源附近或镜头边缘更明显。
> - **不要**为了“电影感”把高光全部压死、把暗部全部抬灰。高光仍需层次，黑位仍需力量。
> - **不要**让肤色被全局冷暖分离污染。电影化调色依赖肤色保护。
> - **不要**忽视位深与色带问题。重度渐变、雾化、柔光、天空和肤色过渡最好在更高位深下完成，否则容易 banding。
> - **不要**把合成统一仅理解成“调颜色一致”。若光向、透视、景深、噪点频率和锐度关系不一致，色调统一也会显假。#constraints

> [!tip] 因果校验
> - 亮部过于硬 → 需要高光 roll-off 与 Bloom 逻辑
> - 数字画面过于无机 → 需要受控瑕疵
> - 画面太平 → 需要 AO、局部对比、景深、空气透视
> - 材质不可信 → 需要重新组织高光形状、粗糙度与边缘光
> - 风格不稳定 → 需要 Pass 化、分层、非破坏式流程
