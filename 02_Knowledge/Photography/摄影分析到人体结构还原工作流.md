# 摄影分析到人体结构还原工作流
Last_Updated: 2026-05-23

## Overview
- Purpose: 将二维摄影图像逆向还原为三维人体结构，支持绘画分析、角色设计和姿态修正。
- Scope: Covers 焦段判定、镜头畸变、视平线、骨骼地标、透视网格、体块概括和常见错误修正。Does not cover 医学诊断或精确人体测量。

## Core Concepts
- **摄影逆向分析**: 从二维照片推断相机位置、焦距倾向、透视关系和空间压缩。
- **视平线**: 观者眼高在画面中的投射，决定人物与环境的透视关系。
- **骨骼地标**: 锁骨、胸骨、髂前上棘、膝盖、踝、肘等可见或可推断的结构点。
- **内部透视网格**: 在人体体块上建立局部轴线和截面方向，而不只依赖背景透视。
- **素体化**: 将真实人体转化为胸腔、骨盆、四肢、头颈等简化体块。
- **透视短缩**: 肢体朝向镜头时长度在画面上缩短，但体积和截面关系仍需成立。

## Consolidated Principles
- P1 [Critical]: 还原结构前必须先判断镜头和透视，不能直接按二维长度画人体。
- P2 [Critical]: 骨盆与胸腔是人体结构还原的核心，四肢必须挂接在它们的空间方向上。
- P3 [High]: 焦段会改变比例感，广角夸张近大远小，长焦压缩空间。
- P4 [High]: 关键骨骼地标比表面衣褶和轮廓更可靠。
- P5 [High]: 被遮挡结构应根据对称、连接和透视推断，但要标注不确定性。
- P6 [Normal]: 错误修正应先改体块方向，再改外轮廓。

## Mechanisms & Logic
- Flow: 1. 判断镜头和畸变 -> 2. 定位视平线和机位 -> 3. 找骨骼地标 -> 4. 建内部透视网格 -> 5. 还原胸腔骨盆 -> 6. 接入四肢和头颈 -> 7. 检查重心。
- Decision Rules: 肢体看似过短 -> 判断是否为透视短缩，不要直接拉长。
- Decision Rules: 姿态不稳 -> 检查重心垂线是否落在支撑面内。
- Decision Rules: 服装遮挡身体 -> 用褶皱张力和边缘厚度辅助判断体块。
- Decision Rules: 背景线条畸变 -> 先校正镜头畸变再分析人体比例。

## Constraints
- 不要把照片外轮廓当作真实解剖边界。
- 不要忽略镜头广角造成的头、手、脚放大。
- 不要用平均人体比例覆盖极端机位下的透视变化。
- 不要把推测部分写成确定事实。
