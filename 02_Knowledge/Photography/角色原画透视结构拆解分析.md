# 角色原画透视结构拆解分析
Last_Updated: 2026-05-23

## Overview
- Purpose: 蒸馏角色原画的透视结构拆解方法，把虚拟镜头、人体工程、衣褶装备和综合工作流统一成分析框架。
- Scope: Covers 焦距侦测、机位解码、视锥控制、人体模型化、解剖地标、装备衣褶、常见误区和整合流程。Does not cover 官方资产复刻或精确医学解剖。

## Core Concepts
- **虚拟镜头**: 原画虽然不是照片，但仍模拟镜头位置、焦距、视角和空间压缩。
- **视锥控制**: 画面中近大远小、边缘拉伸和短缩强度共同提示镜头宽窄。
- **人体模型化**: 将有机人体拆成几何体，便于判断朝向、旋转和遮挡。
- **解剖地标系统**: 用肩峰、锁骨、肋弓、髂嵴、膝、踝、肘等点锁定结构。
- **衣褶与装备逻辑**: 表面元素必须服从体块、重力、张力和挂点。
- **整合工作流**: 透视、结构、表面、光影和叙事焦点应逐层验证。

## Consolidated Principles
- P1 [Critical]: 角色原画的透视拆解必须先确定虚拟镜头，再还原人体结构。
- P2 [Critical]: 不能用平面轮廓替代三维体块分析。
- P3 [High]: 广角冲击应伴随可解释的近大远小和边缘变形。
- P4 [High]: 解剖地标是衣服、护甲和特效之下的结构锚点。
- P5 [High]: 装备厚度、搭接和遮挡关系决定设计可信度。
- P6 [Normal]: 常见错误应映射到透视、体块、重心或表面逻辑，而不是泛称画错。

## Mechanisms & Logic
- Flow: 1. 识别虚拟焦距和机位 -> 2. 定位视平线和消失方向 -> 3. 建立胸腔骨盆头部 -> 4. 接入四肢短缩 -> 5. 分析衣褶装备挂点 -> 6. 检查光影与空间一致性。
- Decision Rules: 如果动作夸张但不可信，检查重心、支撑点和短缩截面。
- Decision Rules: 如果装备像贴纸，增加厚度、重叠、投影和挂点。
- Decision Rules: 如果透视混乱，分离背景透视和人体局部透视。
- Decision Rules: 如果角色不立体，强化体块明暗和边缘重叠。

## Constraints
- 不要用单一消失点强行解释所有有机结构。
- 不要忽略风格化原画中允许的透视夸张，但夸张仍需服务叙事。
- 不要将原画分析转化为受保护角色复制流程。
- 不要把光效、粒子和披风当成结构主体。
