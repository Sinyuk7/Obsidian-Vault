# 漫画身材美学 比例与心理
Last_Updated: 2026-05-23

## Overview
- Purpose: 解释漫画身材美学如何通过比例、心理暗示和结构变形形成角色吸引力。
- Scope: Covers 超常刺激、小脸文化、头身比、躯干流线、骨盆与腰部形态、心理感知边界。Does not cover 身体评价标准化或不安全性化内容。

## Core Concepts
- **身体作为视觉符号**: 漫画身材不是现实复制，而是把力量、优雅、速度、可爱或成熟等特征编码到比例中。
- **小脸效应**: 头部相对缩小会让身体显得更高、更成熟或更时尚。
- **头身比跃迁**: 从 6 头身到 12 头身不是连续写实，而是从现实、商业理想、英雄化到超现实符号的变化。
- **流体躯干**: 躯干不只是桶形体块，而是胸腔、腰部和骨盆之间的连续转折。
- **骨盆架子理论**: 骨盆决定腰臀线的支撑，不能只用外轮廓画曲线。
- **心理放大**: 观众对腿长、腰臀差、肩颈线和姿态张力有高敏感度。

## Consolidated Principles
- P1 [Critical]: 比例夸张必须先确定角色年龄、类型和风格层级。
- P2 [Critical]: 性化比例不得与幼态符号、未成年人身份或模糊年龄结合。
- P3 [High]: 小脸和长腿会互相放大，需要用肩宽、手脚尺寸和躯干体积平衡。
- P4 [High]: 腰部曲线来自胸腔与骨盆的相对关系，不是简单向内收线。
- P5 [High]: 视觉吸引来自节奏和对比，而不是所有部位同时夸张。
- P6 [Normal]: 不同风格目标需要不同夸张强度，现实摄影应更克制。

## Mechanisms & Logic
- Flow: 1. 设定风格坐标 -> 2. 选头身比 -> 3. 分配头、躯干、腿比例 -> 4. 设计肩腰髋节奏 -> 5. 检查骨架和重心 -> 6. 调整材质真实度。
- Decision Rules: 如果脸过小导致违和，降低身体拉伸或提高插画化程度。
- Decision Rules: 如果曲线显假，增加肋骨、腹部和骨盆之间的体积过渡。
- Decision Rules: 如果性感与可爱冲突，明确主导角色定位。

## Constraints
- 不要将风格化比例当作现实身体优劣判断。
- 不要用幼态头脸配成人性感身体作为目标。
- 不要让夸张比例破坏动作可执行性。
- 不要把心理效应写成绝对审美事实。
