# 角色设计体块与素体重建
Last_Updated: 2026-05-23

## Overview
- Purpose: 整理角色设计中从动态手势到体块素体的重建方法，使姿态、透视和结构在绘制前成立。
- Scope: Covers 动作线、节奏、胸腔、骨盆、肢体、关节、短缩、重心、错误诊断和数字素体。Does not cover 完整 3D 建模教程或医学解剖。

## Core Concepts
- **素体重建**: 将复杂角色还原为可检查的基础人体体块，服务姿态、透视和装备设计。
- **动作线**: 概括身体主要能量方向，是姿态设计的第一层。
- **节奏与流动**: 大形曲线和反曲线让体块连接产生生命感。
- **胸腔体块**: 可用卵形或盒形概括，决定上身朝向和肩线。
- **骨盆体块**: 可用内裤形、梯形盒或楔形理解，决定重心和腿部挂接。
- **短缩**: 朝向镜头的体块需要通过截面、重叠和明暗表达体积。

## Consolidated Principles
- P1 [Critical]: 角色设计必须先让素体成立，再叠加服装、装备和风格化细节。
- P2 [Critical]: 动作线不能替代体块；流动感必须落在胸腔、骨盆和四肢结构上。
- P3 [High]: 胸腔和骨盆的扭转、倾斜和压缩决定姿态动态。
- P4 [High]: 关节是体块转换区，不能被画成简单折线。
- P5 [High]: 重心垂线和支撑面决定站姿是否可信。
- P6 [Normal]: 数字素体或 3D 参考可以辅助，但不能替代理解结构。

## Mechanisms & Logic
- Flow: 1. 画动作线 -> 2. 放胸腔和骨盆 -> 3. 建立头颈方向 -> 4. 接入四肢体块 -> 5. 标关节和截面 -> 6. 检查重心 -> 7. 添加服装装备。
- Decision Rules: 姿态僵硬 -> 增加胸腔与骨盆的对抗和压缩拉伸。
- Decision Rules: 肢体扁平 -> 增加截面椭圆、重叠和透视缩短。
- Decision Rules: 角色站不稳 -> 调整支撑脚、骨盆位置和重心线。
- Decision Rules: 细节遮结构 -> 暂时隐藏服装装备，回到素体检查。

## Constraints
- 不要直接在复杂服装上找姿态。
- 不要让风格化比例破坏关节功能和重心。
- 不要用 3D 模型机械描摹导致动作僵硬。
- 不要忽略手脚作为透视和重心终端的作用。
