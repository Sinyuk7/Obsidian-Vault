# 伯里曼人体结构教学精华总结
Last_Updated: 2026-05-23

## Overview
- Purpose: 提炼 George Bridgman 人体结构教学中的艺用建构逻辑，用于绘画、摄影姿态分析和角色设计。
- Scope: Covers 体块化、观念先行、楔入、互锁、韵律、重力、手脚结构和衣褶逻辑。Does not cover 医学解剖细节或逐页教材复述。

## Core Concepts
- **建构解剖**: 不追求医学命名完整，而是把人体理解为可受力、可旋转、可透视的体块系统。
- **观念先行**: 画人体前先建立三维体块观念，再处理表面轮廓。
- **体块化**: 胸腔、骨盆、头、四肢被概括为盒、楔、柱、球等可透视形体。
- **楔入**: 人体部件通过相互嵌入和挤压连接，避免像松散零件。
- **互锁**: 肌肉与骨骼结构相互咬合，形成力的连续传递。
- **韵律**: 曲线、反曲线和力线让人体从静态结构变成有方向的运动系统。

## Consolidated Principles
- P1 [Critical]: 人体不能只画外轮廓，必须先理解内部体块和受力关系。
- P2 [Critical]: 胸腔与骨盆的相对倾斜决定姿态可信度。
- P3 [High]: 关节不是折点，而是体块之间的过渡、压缩和互锁区域。
- P4 [High]: 韵律线必须服务重力和结构，不能变成装饰曲线。
- P5 [High]: 手脚是结构表达的终端，不能被当作附属细节。
- P6 [Normal]: 衣褶应服从身体体块、张力点和重力方向。

## Mechanisms & Logic
- Flow: 1. 找动作线 -> 2. 放置胸腔与骨盆 -> 3. 建立头颈连接 -> 4. 插入四肢体块 -> 5. 检查楔入和互锁 -> 6. 加表面肌肉和衣褶。
- Decision Rules: 姿态僵硬 -> 检查胸腔、骨盆和重心是否形成对抗。
- Decision Rules: 肢体漂浮 -> 增加关节体块、楔入关系和压缩面。
- Decision Rules: 轮廓不可信 -> 回到盒体透视和截面方向。
- Decision Rules: 衣褶混乱 -> 找支撑点、拉伸点和受压点。

## Constraints
- 不要从皮肤轮廓直接推断结构。
- 不要把人体简化为火柴人后直接加肌肉。
- 不要让左右两侧完全对称，真实姿态通常有压缩与拉伸对比。
- 不要牺牲重心平衡追求夸张曲线。
