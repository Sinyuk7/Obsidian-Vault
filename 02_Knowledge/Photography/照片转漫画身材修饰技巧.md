# 照片转漫画身材修饰技巧
Last_Updated: 2026-05-23

## Overview
- Purpose: 整理将照片转向漫画式身材观感时的结构修饰原则，兼顾视觉夸张和写实可信度。
- Scope: Covers 头身比重定义、S 曲线、Puppet Warp、Liquify、Dodge & Burn、局部结构调整和失败防范。Does not cover 非自愿修图、身体贬损或未成年人性化。

## Core Concepts
- **感知真实**: 修饰目标不是数学真实，而是让观者感到比例、姿态和体积自然。
- **漫画化身材**: 通过小头、长腿、细腰、明确肩颈线和腰臀节奏产生风格化吸引。
- **Puppet Warp**: 适合调整骨架方向和大段肢体位置。
- **Liquify**: 适合软组织轮廓微调，不适合承担大幅骨架重排。
- **Dodge & Burn**: 用局部明暗塑造体积，避免只靠轮廓变形。
- **纹理保持**: 漫画化不等于磨平皮肤，真实纹理能支撑可信度。

## Consolidated Principles
- P1 [Critical]: 先修骨架和姿态，再修轮廓和皮肤。
- P2 [Critical]: 不要把真实照片直接拉到极端漫画比例，否则会出现解剖和背景矛盾。
- P3 [High]: 腿长感来自腰线、膝盖、脚踝、鞋型和相机裁切的共同作用。
- P4 [High]: 腰臀曲线必须保留腹部、骨盆和大腿根部的自然过渡。
- P5 [High]: 明暗塑形比单纯收线更能建立漫画式体积。
- P6 [Normal]: 修饰幅度应匹配成片风格，越写实越克制，越插画化越可夸张。

## Mechanisms & Logic
- Flow: 1. 判断原图焦段和姿态 -> 2. 设定目标头身比 -> 3. 用 Puppet Warp 调整骨架 -> 4. 用 Liquify 修局部轮廓 -> 5. 用明暗重塑体积 -> 6. 修复纹理和背景。
- Decision Rules: 需要长腿时先提升视觉腰线和延展脚踝，不直接拉伸整腿。
- Decision Rules: 需要细腰时保留肋骨、腹部和骨盆转折。
- Decision Rules: 背景变形时用遮罩保护或局部重建。
- Decision Rules: 皮肤糊掉时回到频率分离或纹理层修复。

## Constraints
- 不要在没有授权的真实人物照片上做欺骗性身体改造。
- 不要把未成年人或幼态角色作为性感漫画化目标。
- 不要让手脚、关节、衣纹和背景线条暴露变形痕迹。
- 不要用单一液化步骤完成全部修改。
